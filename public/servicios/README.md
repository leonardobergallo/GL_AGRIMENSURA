# Imágenes de Servicios

Coloca aquí las imágenes reales de cada servicio.

## Formatos recomendados:
- **Tamaño**: 1200x800px
- **Formato**: WebP (mejor) o JPG
- **Peso**: < 200KB optimizado

## Archivos necesarios:
- `mensura.jpg` - Foto de trabajo de mensura en campo
- `usucapion.jpg` - Imagen relacionada a usucapión
- `subdivision.jpg` - Terreno o plano de subdivisión
- `ph.jpg` - Edificio o propiedad horizontal
- `topografia.jpg` - Equipo GPS o estación total
- `amojonamientos.jpg` - Mojones o demarcación de terreno

## Optimización:
Usar herramientas como:
- https://tinypng.com
- https://squoosh.app
- https://imageoptim.com
